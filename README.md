# Chess Maze Game
A basic Maze game where the user can only make moves based on the movement of chess pieces 

## Technologies Used
- C#
- .Net
- Windows Presentation Foundation
- Windows Forms

## Getting Started 

### Prerequisites
- .NET 8.0 SDK (Windows)

### Windows Presentation Foundation
```bash
dotnet run --project ChessMazeGameWPF
```

### Windows Forms
```bash
dotnet run --project ChessMazeGameWindowsForm
```
